Importance Weighted On-Policy Distillation

IW-OPD reallocates token-level teacher supervision toward student-sampled prefixes where the teacher can still provide actionable reasoning guidance.

01 / Overview

Dense OPD supervision is useful, but not every token deserves the same update budget.

On-Policy Distillation replaces sparse trajectory-level rewards with dense token-level supervision from a teacher model. Standard OPD, however, averages per-token KL terms uniformly, assuming every student-sampled prefix is equally useful.

IW-OPD shows this assumption is false: teacher continuation accuracy decreases as cumulative prefix divergence grows. The method treats OPD as a constrained local projection and derives a prefix-dependent importance rule that prioritizes teacher-compatible prefixes.

The practical algorithm uses cumulative prefix KL as a stable, stop-gradient surrogate. It is a lightweight modification to OPD code, requires no extra model evaluations, and improves convergence and final quality in same-size and cross-scale distillation.

02 / Challenge

Student rollouts contain teacher-compatible and drifted prefixes in the same trajectory.

Uniform weighting spends the same gradient budget on both, even though their supervision value is very different.

Uniform OPD Limitation

Standard OPD aggregates token KL terms with equal weight. In long reasoning trajectories, an early student deviation can push later prefixes into regions where the teacher's next-token distribution is no longer a useful correction.

The token-budget, residual-gap, and across-position analyses below are further observations that support this limitation: useful supervision is concentrated, and the residual mismatch remains heterogeneous rather than becoming uniformly useful.

  • Low-divergence prefixes: teacher and student still condition on similar histories.
  • High-divergence prefixes: the student has drifted away, so teacher guidance has less leverage.
  • Finite-step training: each update should spend budget where correction is actionable.

03 / Method

IW-OPD converts prefix compatibility into a stable token-level multiplier.

The ideal weight comes from a constrained projection view; the practical weight keeps the ordering while removing raw likelihood-ratio variance.

From local projection to ideal weight Derivation

OPD is a finite-step update, so we first ask for the teacher-closest distribution that is still reachable from the current student. The constrained projection is

Local target
\[ Q_\theta^\star = \arg\min_Q D_{\mathrm{KL}}(Q \,\|\, P_T), \quad \mathrm{s.t.}\; D_{\mathrm{KL}}(Q \,\|\, P_\theta) \le \rho . \]
Solution
\[ Q_\theta^\star(y) \propto P_\theta(y)^{1-\alpha}P_T(y)^\alpha = P_\theta(y) \left(\frac{P_T(y)}{P_\theta(y)}\right)^\alpha . \]
Ideal prefix weight
\[ w_t^\star(y_{<t}) = \left(\frac{P_T(y_{<t})}{P_\theta(y_{<t})}\right)^\alpha = \prod_{k<t} \left( \frac{\pi_T(y_k \mid y_{<k})} {\pi_\theta(y_k \mid y_{<k})} \right)^\alpha . \]

This gives the desired ordering: prefixes more compatible with the teacher receive more OPD update budget.

Practical surrogate Final rule

The raw prefix likelihood ratio is too high variance, signed log-ratios can cancel earlier mismatches, and absolute prefix drift has different scales across trajectories. IW-OPD therefore keeps only the causal ordering and normalizes within each rollout.

Surrogate prefix drift
\[ \delta_k = \left| \log \pi_T(y_k \mid x,y_{<k}) - \log \pi_\theta(y_k \mid x,y_{<k}) \right|, \qquad C_t=\sum_{k<t}\delta_k . \]
IW-OPD weight and advantage
\[ w_t = 1-\frac{C_t}{\sum_{k=1}^{T}\delta_k+\varepsilon}, \qquad A_t^{\mathrm{IW\!-\!OPD}} = A_t^{\mathrm{OPD}} + w_t A_t^{\mathrm{OPD}} . \]
04 / Results

IW-OPD improves both sample efficiency and final distillation quality.

The gains are largest when the student has limited overlap with the teacher's preferred reasoning region.
